# Kongroo.Templates

Company starter templates for `dotnet new`.

```bash
dotnet new install Kongroo.Templates
dotnet new kongroo-sln -n Kongroo.Billing
```

| Template | Short name |
|---|---|
| Solution | `kongroo-sln` |
| Class library | `kongroo-lib` |
| Web API | `kongroo-api` |
| NuGet package | `kongroo-nuget` |
| Unit tests | `kongroo-test` |
| Integration tests | `kongroo-itest` |
