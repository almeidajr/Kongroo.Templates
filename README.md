# Kongroo.Templates

Opinionated .NET starter templates for my personal projects.

```bash
dotnet new install Kongroo.Templates
dotnet new kongroo-sln -n Billing   # scaffolds Kongroo.Billing.* (Api, UnitTests, ...)
```

| Template | Short name |
|---|---|
| Solution | `kongroo-sln` |
| Class library | `kongroo-lib` |
| Web API | `kongroo-api` |
| NuGet package | `kongroo-nuget` |
| Unit tests | `kongroo-test` |
| Integration tests | `kongroo-itest` |
